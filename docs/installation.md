# Installation

## Stable release

To install Pyenspp, run this command in your terminal:

```sh
uv add pyenspp
```

Or if you prefer to use `pip`:

```sh
pip install pyenspp
```

## From source

The source files for Pyenspp can be downloaded from the [Github repo](https://github.com/Curallin/pyenspp).

You can either clone the public repository:

```sh
git clone https://github.com/Curallin/pyenspp
```

Or download the [tarball](https://github.com/Curallin/pyenspp/tarball/main):

```sh
curl -OJL https://github.com/Curallin/pyenspp/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd pyenspp
uv sync
```
